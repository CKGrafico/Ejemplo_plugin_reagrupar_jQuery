####Descripción:
Veamos un ejemplo:
Tenemos una lista de zapatos, para mostrarlos hacemos una consulta a nuestra base de datos y mostramos los resultados.

Nos gustaría poder agrupar los zapatos por: color, precio, talla.

Cuando hacemos la consulta a la BD podemos escoger una de las anteriores, por ejemplo por talla.

Si queremos ordenarlos de otra manera, normalmente ponemos un botón que al
clicar genera de nuevo la consulta y nos los muestra como queremos.
